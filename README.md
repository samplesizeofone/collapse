# Overview
Guided origami crease pattern folding using [trilateration](https://en.wikipedia.org/wiki/Trilateration).

# Goals
Collapse the [diamond base](http://www.linkless.org/diamond-base/) into its folded form with arbitrary detail.

# Stretch goal
Animate the folding of a field of cranes out of collapsed diamond bases.

# Setup
